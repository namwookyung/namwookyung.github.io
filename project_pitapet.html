본인 디자인 화면

메인 화면

상단바와 푸터에 주요색 사용 / 배너 3가지 자동 순차 반복 / 베스트 아이템 3가지 배치

회원가입 화면

아이디, 비밀번호, 휴대폰번호, 이메일, 주소 기입란 / 가입, 다시 작성(초기화) 버튼 하단 배치

로그인 화면

아이디, 비밀번호 기입란 / 로그인 버튼 하단 배치

장바구니 화면

상품별 열 구분, 열 좌측에서 우측으로 번호, 상품명, 가격, 수량 조절 버튼 / 좌측 하단 삭제 버튼 / 우측 하단 총 결제 금액 / 중앙 하단 쇼핑 계속하기 문구 배치

관리자 메인

상단바는 보조색을 사용해 이용자 화면과 색상으로 차이를 둠

회원 목록

가입 회원을 보기 쉽게 한줄씩 중앙 정렬 / 회원 아이디 각 우측 삭제 문구 배치

회원 정보 상세

회원 상세 정보 / 리스트로 돌아가기(회원 목록) 문구 우측 하단 배치

공지사항

1페이지 당 10개씩 보이도록 배치

공지사항 관리

1페이지 당 10개씩 보이도록 배치

공지사항 등록

제목, 내용 기입란 배치 후 하단 파일 첨부 / 다시 작성(초기화), 목록 버튼 하단 배치

공지사항 상세

제목, 첨부파일, 내용 내림차순 배치 / 수정, 삭제, 목록 버튼 우측 하단 배치

공지사항 수정

제목, 내용을 등록할 때와 동일하게 배치 / 수정, 뒤로(취소) 버튼 하단 배치

본인 개발 기능

회원가입

아이디, 비밀번호, 휴대폰번호, 이메일, 주소 기입 후 가입 기능 / 다시 작성 클릭 시 기입란 초기화 기능

로그인

아이디, 비밀번호 기입 후 로그인 기능

장바구니

상품 삭제, 상품 수량 증감 조절, 상품 총 금액 합산 표기 기능

공지사항

1페이지 당 10개씩 보이도록 설정 / 게시글 클릭 시 상세 보기 기능

공지사항 관리

1페이지 당 10개씩 보이도록 설정 / 게시글 클릭 시 상세 보기 가능

공지사항 등록

제목과 내용 기입, 파일 첨부 후 등록 버튼 클릭 시 게시글 등록 / 다시 작성 클릭 시 기입란 초기화 / 목록 보기 클릭 시 공지사항 목록 페이지로 이동

공지사항 상세

수정 클릭 시 공지사항 수정 페이지로 이동 / 삭제 클릭 시 해당 게시글 삭제 가능

공지사항 수정

제목, 내용 수정 후 수정 클릭 시 해당 부분 수정 가능
